The **Developer Hub** is a unified interface that brings together infrastructure and services needed for building, deploying, and managing AI/ML applications. It simplifies the creation of resource instances using predefined compute or service profiles.

## What You'll Find Here

The TELUS AI Factory Developer Hub provides comprehensive tools and services organized into two main categories:

### Compute Resources

High-performance computing infrastructure for your ML workloads:

- **[Bare Metal](/docs/compute/bare-metal)** - Dedicated servers with full hardware access
- **[Virtual Machines](/docs/compute/virtual-machines)** - Flexible, isolated environments
- **[Kubernetes](/docs/compute/kubernetes)** - Efficient container orchestration

### Services

Managed services to accelerate your AI/ML development:

- **[Notebooks](/docs/services/notebooks)** - Interactive development environments
- **[vLLM Inference Services](/docs/services/vllm)** - Optimized LLM serving
- **[Ollama Inference Services](/docs/services/ollama)** - Local LLM deployment
- **[NIM Services](/docs/services/nim)** - NVIDIA Inference Microservices
- **[Marketplace Applications](/docs/services/marketplace)** - Pre-configured ML solutions

## Key Features

- **Unified Interface**: Access all compute and service resources from a single platform
- **Predefined Profiles**: Quick setup with optimized configurations
- **Scalable Resources**: From single machines to full Kubernetes clusters
- **ML-Optimized**: Purpose-built for AI/ML workloads
- **Interactive Tools**: Notebook environments for data analysis and model development
